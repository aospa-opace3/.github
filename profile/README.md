# aospa-opace3

A project to port AOSPA beryl to OnePlus Ace 3.

## Project Details

* **Source:** https://github.com/AOSPA-Mod
* **Target Device:** OnePlus Ace 3 (`astonc`)
* **Build Status:** **Bootable** 

## Current State

The system can boots to the home screen.
